CSS transform Property

CSS 2D Transformations

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

And they make an element position relative.

              transform: none|transform-functions|initial|inherit;

translate

The translate() method moves an element from its current position (according to the parameters given for the X-axis and the Y-axis).

translate(x,y) - Defines a 2D translation

translateX(x) - Defines a translation, using only the value for the X-axis

translateY(y) - Defines a translation, using only the value for the Y-axis

scale

The scale() method increases or decreases the size of an element (according to the parameters given for the width and height).

scale(x,y) - Defines a 2D scale translation

scaleX(x) - Defines a scale translation, using only the value for the X-axis

scaleY(y) - Defines a scale translation, using only the value for the Y-axis

Rotate

The rotate() method rotates an element clockwise or counter-clockwise according to a given degree.

rotate(angle) - Defines a 2D rotation, the angle is specified in the parameter

rotateX(angle) - Defines a 3D rotation along the X-axis

rotateY(angle) - Defines a 3D rotation along the Y-axis

skew

The skew() method skews an element along the X and Y-axis by the given angles.

skew(x-angle, y-angle) - Defines a 2D skew transformation along the X- and the Y-axis

skewX(angle) - Defines a 2D skew transformation along the X-axis

skewY(angle) - Defines a 2D skew transformation along the Y-axis
