HTML DOM Methods and Properties

Finding HTML Elements

document.getElementById(id) method find an element by element id.

document.getElementsByTagName(name) method find elements by tag name and returns an HTMLCollection object.

HTMLCollection object is an array-like list (collection) of HTML elements. The elements in the collection can be accessed by an index number and the index starts at 0. The length property defines the number of elemets in an HTMLCollection. Note: An HTMLCollection may look like an array, but it is not.

document.getElementsByClassName(name) method find elements by class name.

document.querySelectorAll(CSS selector) method find all HTML elements that matach a specified CSS selector and return a NodeList object.

NodeList object is a list (collection) of nodes extracted from a document. It is almost the same as HTMLCollection object. The elements in the NodeList can be accessed by an index number and the index starts at 0. The length property defines the number of nodes in a node list.

querySelector vs. getElementById

Both the querySelector() and getElementById() methods let you retrieve an element from the JavaScript Document Object Model (DOM). However, each method has its own user cases.

The JavaScript querySelector() method lets you retrieve an element from the DOM, or the web page, using a CSS selector. This method comes with a sister function called querySelectorAll() which selects all the elements that match a particular selector from the DOM.

The getElementById() method retrieves an element based on its ID attribute. You would use this method if you only want to retrieve one element from the web page.

You should opt to use the querySelector() method if you need to select elements using more complex rules that are easily represented using a CSS selector.

If you want to select an element by its ID, using getElementById() is a good choice.

In Conclution,they both are the same. Just be careful with querySelectorAll() method, as it returns a Node List and you will have to select the node before DOM Manupulation.

Sets or Returns HTML Elements

element.style.property property change the style of an HTML element.

element.setAttribute(attribute, value) method change the attribute value of an HTML element.

Adding and Deleting Elements

document.createElement(element) method create an HTML element.

document.createTextNode(Text) method creates a text node.

document.removeChild(element) method removes an HTML element.

document.appendChild(element) method adds an HTML element as the last child of the parent.

document.insertBefore(before element, element) method adds an HTML element before the targeted element.

document.replaceChild(new, old) method replace an HTML element.

document.write(text) method write into the HTML output stream.

Event Listener

addEventListener() method attaches an event handler to the specified element.

removeEventListener() method removes event handlers that have been attached with the addEventListener() method.

JavaScript String Methods

  1. string.length - returns the length of a string
  2. slice(start, end) - extracts a part of a string and returns the extracted part in a new string.
  3. substring(start,end) - similar to slice() but cannot accept negative indexes.
  4. substr() - similar to slice(), the difference is that second parameter specifies the length of the extracted part.
  5. replace() - replaces a specified value with another value in a string.
  6. toUpperCase() - A string is converted to upper case.
  7. toLowerCase() - A string is converted to lower case.
  8. concat() - joins tow or more strings.
  9. trim() - removes whitespace from both sides of a string.
  10. charAt(position) - returns the character at a specified index (position) in a string.
  11. charCodeAt() - returns the unicode of the character at a specified index in a string.
  12. property access [] - ECMAScript 5 (2009) allows property access [] on strings.
  13. split() - A string can be converted to an awway with the split() method.

String Search Methods

  1. indexOf() - returns the index of (the position of) the first occurance of a specified text in a string.
  2. lastIndexOf() - returns the index of the last occurance of a specified text in a string.
  3. search() - searches a string for a specified value and returns the position of the match.
  4. match(regexp) - searches a string for a match against a regular expression, and returns the matches, as an Array object.
  5. includes() - returns true if a string contains a specified value.
  6. startsWith() - returns true if a string begins with a specfied value, otherwise false.
  7. endsWith() - returns true if a string ends with a specified value, otherwise false.

Note: All string methods return a new string. They don't modify the original string.

Formally said: Strings are immutable: Strings cannot be changed, only replaced.

JavaScript Number Methods

Number methods help you work with numbers.

Primitive values (like 3.14 or 2014), cannot have properties and methods (because they are not objects).

But with JavaScript, methods and properties are also available to primitive values, because JavaScript treats primitive values as objects when executing methods and properties.

  1. toString() - method returns a number as a string.
  2. toExponential() - returns a string, with a number rounded and written using exponential notation.
  3. toFixed() - returns a string, with the number written with a specified number of decimals.
  4. toPrecision() - returns a string, with a number written with a specified length.
  5. valueOf() - returns a number as a number.

In JavaScript, a number can be a primitive value (typeof = number) or an object (typeof = object).

The valueOf() method is used internally in JavaScript to convert Number objects to primitive values.

Converting Variables to Numbers

There are 3 JavaScript methods that can be used to convert variables to numbers:

  1. Number() - returns a number, converted from its argument.
  2. parseFloat() - parses its argument and returns a floating point number.
  3. parseInt() - parses its argument and returns an integer.

If the number cannot be converted, NaN (Not a Number) is returned.

Number Properties

  1. MAX_VALUE - returns the largest possible number in JavaScript.
  2. MIN_VALUE - returns the lowest possible number in JavaScript.
  3. POSITIVE_INFINITY - represents infinity (returned on overflow).
  4. NEGATIVE_INFINITY - represents negative infinity (returned on overflow).
  5. NaN - represents a "Not-a-Number" value.

JavaScript Array Methods

  1. toString() - method converts an array to a string of (comma separated) array values.
  2. join() - method joins all array elements into a string. It behaves like toString(), but in addition you can specify the separator.
  3. pop() - method removes and returns the last element from an array.
  4. push() - method adds a new element to an array (at the end) and returns the new array length.
  5. shift() - method removes the first array element and "shifts" all other elements to a lower index.
  6. unshift() - method adds a new element to an array (at the beginning) and returns the new array length.
  7. length - length property provides an easy way to append a new element to an array.
  8. delete - delete operator deletes array elements and leaves undefined holes in the array.
  9. concat() - method creates a new array by merging (concatenating) existing arrays and returns a new array.
  10. splice() - method can be used to add new items to an array and returns an array with the deleted items.
  11. slice() - method slices out a piece of an array into a new array and does not remove any elements from the source array.

Array Sorting Methods

  1. sort() - method sorts an array alphabetically.
  2. reverse() - method reverses the elements in an array.

Numeric Sort

By default, the sort() function sorts values as strings.

The sort() method will produce incorrect result when sorting numbers.

You can fix this by providing a compare function.

              const points = [40, 100, 1, 5, 25, 10];
              points.sort(function(a, b){return a - b});

Use the same trick to sort an array decending.

              const points = [40, 100, 1, 5, 25, 10];
              points.sort(function(a, b){return b - a});

Using Math.max() on an Array

You can use Math.max.apply to find the highest number in an array.

              function myArrayMax(arr) {
               return Math.max.apply(null, arr);
              }

You can use Math.min.apply to find the lowest number in an array.

            function myArrayMax(arr) {
             return Math.min.apply(null, arr);
            }

Array Iteration Methods

Array forEach() method

The forEach() method calls a function (a callback function) once for each array element.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

              const numbers = [45, 4, 9, 16, 25];
              let txt = "";
              numbers.forEach(myFunction);

              function myFunction(value, index, array) {
                 txt += value + "</br>";
              }

Array map() method

The map() method creates a new array by performing a function on each array element.

The map() method does not execute the function for array elements without values.

The map() method does not change the original array.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

Array filter() method

The filter() method creates a new array with array elements that passes a test.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

Array reduce() method

The reduce() method runs a function on each array element to produce (reduce it to) asingle value.

The reduce() method works from left-to-right in the array.

The reduce() method does not reduce the original array.

Note that the function takes 4 arguments: the total (the initial value/ previously returned value), the item value, the item index and the array itself.

The reduce() method can accept an initial value.

The reduceRight() works from right-to-left in the array.

Array every() method

The every() method checks if all array values pass a test.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

Array some() method

The some() method check if some array values pass a test.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

Array indexOf() method

The indexOf() method searches an array for an element value and returns its position.

Note: The first item has position 0, the second item has position 1, so on.

              array.indexOf(item, start)

item -> Required. The item to search for.

start -> Optional. Where to start the search.

Array.indexOf() returns -1, if the item is not found.

If the item is present more than once, it returns the position of the first occurance.

Array.lastIndexOf() is the same as Array.indexOf(), but returns the position of the last occurance of the specified element.

Array find() method

The find() method returns the value of the first array element that passes a test function.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

Array findIndex() method

The findIndex() method returns the index of the first array element that passes a test function.

Note that the function takes 3 arguments: the item value, the item index and the array itself.

Array.from() method

The Array.from() method returns an Array object from any object with a lenght property or any iterable object.

The keys() method

The Array.keys() method returns an Array Iterator object with the keys of an array.

They includes() method

The Array.includes() method allows us to check if an element is present in an array (including NaN).

Array.includes() allows to check for NaN values. Unlike Array.indexOf().

JavaScript Date Methods

When a Date object is created, a number of methods allow you to operate on it.

Date methods allow you to get and set the year, month, day, hour, minute, second, and millisecond of date object, using either local time or UTC (unversal or GMT) time.

JavaScript will (by default) output dates in full text string format.

When you display a date object in HTML, it is automatically converted to a string, with the toString() method.

The toUTCString() method converts a date to a UTC string (a date disply standard).

The toDateString() method converts a date to a more readable format.

The toISOString() method converts a Date object to a string, using the ISO standard format.

The Date.parse() returns the number of milliseconds between the date and January 1, 1970.

Date Get Methods

The getTime() method returns the number of milliseconds since January 1, 1970.

The getFullYear() method returns the year of a date as a four digit number.

The getMonth() method returns the month of a date as a number (0 - 11). You can use an array of names, and getMonth() to return the month as a name.

The getDate() method returns the day of a date as a number (1 - 31).

The getHours() method returns the hours of a date as a number (0 - 23).

The getMinutes() method returns the miutes of a date as a number (0 - 59).

The getSeconds() method returns the seconds of a date as a number (0 - 59).

The getMilliseconds() method returns the milliseconds of a date as a number (0 - 999).

The getDay() method returns the weekday of a date as a number (0 - 6).

In JavaScript, dthe first day of the week (0) means "Sunday", even if some countries in the world consider the first day of the week to be "Monday".

Date Set Methods

The setFullYear() method sets the year of a date object and can optionally set month and day.

The setMonth() method sets teh month of a date object (0 - 11).

The setDate() method sets the day of a date object (1 - 31).

The setHours() method sets the hours of a date object (0 - 23).

The setMinutes() method sets the minutes of a date object (0 - 59).

The setSeconds() method sets the seconds of a date object (0 - 59).

JavaScript Math Methods

The Math object is static. All methods and properties can be used without creating a Math object first.

The syntax for any Math property is Math.property.

JavaScript provides 8 mathematical contants that can be accessed as Math properties.

  1. Math.E returns Euler's number.
  2. Math.PI returns PI.
  3. Math.SQRT2 returns the square root of 2.
  4. Math.SQRT1_2 returns the square root of 1/2.
  5. Math.LN2 returns the natural logarithm of 2.
  6. Math.LN10 returns the natural logarithm of 10.
  7. Math.LOG2E returns base 2 logarithm of E.
  8. Math.LOG10E returns base 10 logarithm of E.

Number to Integer

There are 4 common methods to round a number to an integer.

  1. Math.round(x) returns x rounded to its nearest integer.
  2. Math.ceil(x) returns x rounded up to its nearest integer.
  3. Math.floor(x) returns x rounded down to its nearest integer.
  4. Math.trunc(x) returns the integer part of x.

Math.sign(x) returns if x is negative, positive or null.

Math.pow(x, y) returns the value of x to the power of y.

Math.sqrt(x) returns the square root of x.

Math.abs(x) returns the absolute (positive) value of x.

Math.sin(x) returns the sine (a value between -1 and 1) of the angle x (given in radians).

Math.cos(x) returns the cosine (a value between -1 and 1) of the angle x (given in radians).

Math.min() and Math.max() can be used to find the lowest or highest value in a list of arguments.

Math.random() returns a random number between 0 (inclusive), and 1 (exclusive).

Math.log(x) returns the natural logarithm of x.

Math.log2(x) returns the base 2 logarithm of x.

Math.log10(x) returns the base 10 logarithm of x.

Math.random() Method

Math.random() returns a random number between 0 (inclusive), and 1 (exclusive).

Math.random() always returns a number lower than 1.

Math.random() and Math.floor() can be used to return random integers.

              // Returns a random integer from 0 to 9:
              Math.floor(Math.random() * 10);
            
              // Returns a random integer from 1 to 10:
              Math.floor(Math.random() * 10) + 1;

JavaScript Cookies

Cookies let you store user information in web pages.

Cookies are data, stored in small text files, on your computer.

When a web server has sent a web page to a browser, the connection is shut down, and the server forgets everything about the user.

Cookies were invented to solve the problem "how to remember information about the user":

  • When a user visits a web page, his/her name can be stored in a cookie.
  • Next time the user visits the page, the cookie "remembers" his/her name.

Cookies are saved in name-value pairs like: username: John Doe
